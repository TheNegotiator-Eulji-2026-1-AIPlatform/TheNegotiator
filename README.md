# The Negotiator
협상가(2026-1 인공지능플랫폼 팀프로젝트 언리얼 &amp; FastAPI)

## 🛠 Tech Stack
- **Client**: Unreal Engine 5
- **Server**: FastAPI (Python 3.12+)
- **Communication**: REST API (VaRest)

## 🚀 Quick Start
1. 서버 실행: `python -m uvicorn main:app --reload`
2. 언리얼 실행: `TheNegotiator.uproject` 오픈 후 Play
